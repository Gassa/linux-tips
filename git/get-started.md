1. Installation


Please refer to the [guide](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git).

2. Configure


```bash
git config --global user.name "John Doe"
git config --global user.email johndoe@example.com
```


3. Initiate a git project:

```
mkdir PROJECT_NAME
cd PROJECT_NAME
git init
```
